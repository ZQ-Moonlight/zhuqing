### 🛠️ 最近界面更新 / Recent UI Update

- 首页左侧三个入口已重构为轻量胶囊链接，增加横向间距、边框反馈与箭头提示，让「查看作品 / 阅读档案 / 留言给我」更像可点击入口，同时避免橙色按钮过强。
- 首页语言切换已扩展为完整界面状态：切换到 English 后，导航、姓名区、CTA、语言标签、留言弹窗、表单占位符与邮件主题都会显示英文。
- 作品页的主要界面文案已接入同一语言状态；从首页切换到 English 后进入 Works，会优先显示英文导航、项目卡片、工作流入口与分享资料标题。
- 中文模式导航恢复为「中文 / English」双语标签，英文模式仍保持纯英文界面。
- 首页姓名区删除了 BFA、UE5、MetaHuman 等角色标签，并放宽英文姓名副标题行距，避免字形重叠。
- 作品页移除了单独重复出现的《光影长青》技术工作流卡片，保留项目卡片内的入口关系。
